关于实践和训练的重要性

当一个普通人开始接触市场。接触方式往往是读书、参加培训、或者直接拎包就上去了。以上几种人最开始都有一个幸运期。在幸运期赶上趋势，不需要想买的理由，然后把回调扛过来，最后都是赚钱。那么这些赶上幸运趋势的朋友，就会很膨胀，甚至认为在证券市场赚钱是一件十分简单的事。然而，当趋势进入TR行情，或者进入派发阶段之后，他们这种买入死扛的方法会导致严重的亏损甚至爆仓。

 

于是有些人开始反思，然后寻求一些帮助，这些帮助包括参加培训班，或者买更多五花八门的技术分析和基本面分析的书。期望复制书中和培训班中的“成功”方法，不过学习之后，实盘交易还是巨亏。这中间出现了几个问题。

 

第一个问题：碰上滥竽充数的了。中国的证券市场时间比较短，而公众又急于想参与市场，于是有了这种需求，市场产生了诸多的滥竽充数的专家，他们开办三、五天短期速成班诱惑公众，然后讲几个指标讲几个基本面，讲几个平均线，让大家拿几种组合再次回到市场亏损。

 

第二个问题：即使碰上愿意讲一点真东西的人，因为公众太急于回到市场翻本。在没有完全掌握理论的情况下，直接用大资金去交易，最后导致巨额亏损，这反应了在整个学习当中欠缺的一个重要环节——把正确的理论落实到实际的行情中，而理论落地环节需要一名成功的资深交易员手把手的指导，才能避免少走弯路，少在市场中交学费。
