"跳过小河"的故事

时间：2018年12月11日
讲解市场如何突破横盘振荡的市场。在这个故事中，市场被比喻成童子军，横盘振荡的市场被比喻成曲折的小河，上阻力线就是横盘振荡的市场内的反弹高点。当童子军探测到小河的边缘并发现卖盘即将枯竭时，童子军先撤退一段距离，这样方便助跑并"跳过小河"。童子军的力量可以通过振幅和成交量度量出来。

为什么威科夫理论是严肃交易者的选择

2018年12月11日
在交易市场之前，我们应该了解市场的真正含义。 在当今年代，证券市场已经成为一个虚拟的世界。但是无论如何，市场的共性依然存在于证券市场当中。比如某个产品的供求关系决定这价格上涨和下跌。股票也是商品，它的供求关系样决定者它的走势。威科夫强调当需求大于供应的时候，价格处于上涨阶段，反之出现下降阶段。

威科夫交易方法

2018年12月11日
理查德.威科夫是20世纪初研究股票市场的技术方法的先驱。他被认为是技术分析的五大“巨人之一”，以及道琼斯，甘恩，埃利奥特和美林。在15岁的时候，他作为纽约经纪公司的股票选手，在20多岁时成为自己公司的负责人。他成立了“华尔街杂志”，本身即是编辑又是撰稿人。杂志超过20万的用户。威克科夫是一个狂热的市场学生，也是活跃的tape reader和交易员。他仔细观察市场行为以及当时传奇股票经纪人的操盘战役，这些传奇人物包括摩根大通和杰西·利弗莫尔。 从他对这些大师级交易者的观察和访谈中，将他们的操作精华编纂成交易法则，以及交易技巧、资金管理、和交易纪律。

市场技术分析的宗师：理查德.威科夫先生

时间：2018年12月11日
理查德.威科夫（1873年11月2日 - 1934年3月19日）是20世纪初的市场技术分析先驱。他被认为是技术分析的五个“巨头”之一（道琼斯，Gann，Elliott和美林）。 当年和他一起工作的还有：Jesse Livermore（杰西.利佛摩尔），E.H.Harriman，James R.Keene，Otto Kahn，J.P. Morgan，这些人都是当年著名名的市场做手（简称CM）。5岁时，他成为股票经纪人，他20岁时成立了自己经纪公司。在1907年，他创办了当时市场的指南针-“华尔街杂志”，自己既是创办者又是主编，这个刊物有超过20万订阅者。
