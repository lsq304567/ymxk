关于这段时间学习总结

近段时间遇到好多看不懂的，就会造成一种厌烦心理。 等下次回过头来看，就会有一种“原来就是这样的心理”。 总结：应该养成一种遇到问题钻研一会，若是还是有问题，大胆放过，等下次想起来回过头再来看。 宿舍真是一个发霉的地方，相当不适合学习。 给自己找一些好听的歌曲，来降低烦躁心理以及其他人造成的这种垃圾环境。 防止进一步降低自己学习的兴趣去，应该有这么一种心理。 给自己一点进步，就嚣张一点，这样会有一种成就感，加深自己的成就感，以增强自己的学习的兴趣。 平时多去图书馆，也要在手机上随时复习学过的知识，还有不要忘记关于这方面的英语知识单词， 记住这些单词就对事半功倍。

关于软件使用方面的攻略

  1. 音乐：除了天天动听和虾米音乐是在一起合作，其他音乐都是和腾讯合作。 与腾讯合作的音乐软件相对好用的是网易云。
  2. 阅历知识：比较推荐的软件是知乎，果壳。 知乎我使用，感觉想到不错，各种知识相对较全，若是还没有就得到专业网站上去找。 果壳我倒是没有用过，但是有好多人推荐这个。
  3. 浏览器：肯定是无脑Google 但是有人说微软的Edge也不错，可以尝试一下，本人未发现比较好的地方。 可能是我没有发现天赋。
  4. 关于电脑使用方面。 这个就多了，网上有各种简约教程，本人用的是知乎的教程，其他的各种软件都不怎么支持win10. 出现各种bug。 最主要的是：使用电脑要记住各种快捷键方式，千万别自己强设置快捷方式，这样真的很蠢。 要在微软的系统下来有规律设置。
